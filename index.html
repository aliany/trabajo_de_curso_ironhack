The Road to Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broker CSS support, and abandoned.

We must clear the mind of the past. Web enlightenment has been archived thanks to te tirelees efforts of folk like the W3C, WaSP, and the major browser creators.

The CSS Zen Garden invites you to relax and meditate on the important lessons of the masters. Begin to see with clarity. Learn to use the time- honored techniques in new and invigorating fashion. Becomeone with the web.

So What is This About?

There is a continuing need to show the power of CSS . The Zen Garden aims to excite, inspire, and encourage pasticipation. To begin, view some of the existing designs i the list. The HTML remains the same, the only thing that has changed is external CSS file. Yes, really.

CSS allow complete and total control over the style of a hypertext document. The only way this can be illustrated in way that gets people excited is by demostrating what it can truly be, once the reins are place in the hands of those able to create beauty from structure. Designers and coders alike have contributed to the beauty of the web; we can always push it further.
